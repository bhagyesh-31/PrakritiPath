Gamified environmental education and action platform to build daily sustainability habits.
PrakritiPath is a long-term, gamified environmental education and action platform designed to transform sustainability into a daily habit.

## Vision
To build a legacy system where students learn environmental concepts, take real-world actions daily, share verified contributions publicly, and earn authentic, QR-verifiable recognition.

## Core Features
- Flashcards and quizzes for environmental learning
- Daily real-world environmental contributions
- Public user profiles and social following
- Badges, streaks, and state-wise leaderboards
- QR-based verifiable certificates
- Role-based access: students, teachers, admins
